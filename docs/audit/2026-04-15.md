## CRITICAL (5)

### C1. Integer overflow in tensor allocation sizes
- **File**: tensor.cyr:15,26
- **Risk**: Buffer overflow via controlled dimensions. `total * 8` can wrap i64.
- **Fix**: Guard `total` and `total * 8` against overflow before alloc.

### C2. Integer overflow in complex matrix allocation
- **File**: complex.cyr:131
- **Risk**: `rows * cols * 16` wraps → undersized alloc → OOB writes.
- **Fix**: Overflow check before alloc. Same for `cmat_kronecker`.

### C3. Integer overflow in stdlib matrix allocation
- **File**: matrix.cyr:11 (stdlib — needs upstream fix in cyrius)
- **Risk**: `16 + rows * cols * 8` wraps for large inputs.
- **Fix**: Upstream to cyrius stdlib, or add wrapper with checks.

### C4. Integer overflow in Riemann tensor allocation
- **File**: diffgeo.cyr:117
- **Risk**: `dim^4 * 8` overflows for dim >= 55. Even dim=16 is 524K entries.
- **Fix**: Cap dim at 16 (sufficient for all physics applications).

### C5. Sieve unbounded allocation (DoS)
- **File**: num.cyr:476
- **Risk**: `alloc(limit + 1)` with arbitrary `limit` → OOM crash.
- **Fix**: Cap limit at 10M or return error.

## HIGH (10)

### H1. tensor_contract is a no-op (stub)
- **File**: tensor.cyr:147
- **Risk**: Returns zero tensor for all inputs. Any Ricci/trace dependent on this is silently wrong.
- **Fix**: Implement proper multi-index contraction or return error.

### H2. m4_determinant missing fourth cofactor
- **File**: mat4.cyr:168-175
- **Risk**: Wrong result for matrices with non-zero [3][0]. m4_inverse has its own correct determinant.
- **Fix**: Add the m30 cofactor term.

### H3. Division by zero in cx_div/cx_inv
- **File**: complex.cyr:45,69
- **Risk**: Silent NaN/Inf propagation through complex matrix stack, Lie groups.
- **Fix**: Guard denominator near zero.

### H4. Division by zero in dual_div/dual_sqrt/dual_ln
- **File**: autodiff.cyr:45,81,78
- **Risk**: Silent NaN/Inf from forward-mode AD.
- **Fix**: Domain guards.

### H5. Division by zero in world_to_screen
- **File**: transforms.cyr:170
- **Risk**: NaN screen coordinates for points at camera.
- **Fix**: Guard w==0 like m4_transform_point does.

### H6. Division by zero in linearize_depth_reverse_z
- **File**: color.cyr:157
- **Risk**: Inf at far plane.
- **Fix**: Guard ndc==0.

### H7. Division by zero in f64_fmod and f64_tan
- **File**: f64_util.cyr:7,12
- **Risk**: NaN from fmod(x,0), Inf from tan(PI/2).
- **Fix**: Guard y==0 in fmod. Document tan behavior.

### H8. Missing bounds checks on m4_get/m4_set
- **File**: mat4.cyr:29-36
- **Risk**: OOB read/write with invalid col/row.
- **Fix**: Bounds check or document contract.

### H9. BDF-5 coefficient precision mismatch
- **File**: ode.cyr:111-120
- **Risk**: Slightly wrong BDF-5 integration from imprecise hex constants.
- **Fix**: Recompute 300/137 exactly and verify all BDF constants.

### H10. ivl_sin always returns [-1,1] for intervals < PI
- **File**: interval.cyr:115
- **Risk**: Interval arithmetic useless for verified sin bounds.
- **Fix**: Return `ivl_new(lo, hi)` instead of expanding to include -1,1.

## MEDIUM (9)

### M1. num_modpow overflow in intermediate multiplication
- **File**: num.cyr:403-415
- **Risk**: Wrong results for moduli > 2^31.
- **Fix**: Implement mul-mod that avoids overflow (Russian peasant multiplication).

### M2. PCG32 signed arithmetic for unsigned operations
- **File**: num.cyr:320-344
- **Risk**: Potential bias from sign-extending shifts.
- **Fix**: Verify Cyrius shift semantics, add masking if arithmetic shift.

### M3. geo_ray_plane: 0 means both "miss" and "t=0 hit"
- **File**: geo.cyr:221
- **Risk**: Ambiguous API.
- **Fix**: Use negative sentinel for miss.

### M4. Geodesic solver allocates without freeing
- **File**: diffgeo.cyr:307
- **Risk**: Memory growth for long integrations.
- **Fix**: Document, consider scratch space parameter.

### M5. Cholesky division by zero for semi-definite input
- **File**: linalg.cyr (stdlib):388
- **Risk**: NaN from L[j][j]=0.
- **Fix**: Check diag < EPSILON, not just diag < 0.

### M6. GMRES modifies input vector in-place
- **File**: linalg_ext.cyr:364
- **Risk**: Corruption if A_fn returns cached storage.
- **Fix**: Copy wj before modification.

### M7. expr_eval aborts process on undefined variable
- **File**: symbolic.cyr:198
- **Risk**: Library call crashes application.
- **Fix**: Return error/NaN instead of syscall(60,1).

### M8. SVD via A^T*A squares condition number

### M9. Jacobi eigensolver O(n^5) for large matrices
